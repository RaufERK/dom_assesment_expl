## Начало работы

Чекпоинт - это возможность проверить свои собственные знания. Поэтому, 
списывать - бессмысленно!

После клонирования репозитория, перейди в каталог с проектом и выполни
```bash
npm install
```

Тесты запускаются командой `npm test`.

## Release
